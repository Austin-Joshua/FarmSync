## üîê Authentication & Security

### User Registration
- Create account with email, password, and role selection
- Password strength validation (8+ chars, uppercase, lowercase, numbers, symbols)
- Email uniqueness verification
- Automatic account activation upon registration

### User Login
- Secure login with email and password
- Remember session functionality
- Password reset via email
- Automatic session timeout after 30 minutes of inactivity

### Two-Factor Authentication (2FA)
- Optional 2FA setup for enhanced security
- Time-based one-time passwords (TOTP)
- Backup codes for account recovery
- Authenticator app integration (Google Authenticator, Microsoft Authenticator)

### Role-Based Access Control
- **Farmer Role** - Access to farm management, crop tracking, reports
- **Admin Role** - System administration, user management, audit logs
